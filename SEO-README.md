## 🔧 Getting Started

### Prerequisites
- Node.js 18+ or Bun
- PostgreSQL database

### Installation

1. Clone the repository:
```bash
git clone https://github.com/OmarHosamCodes/portfolio.omarhosamcodes.git
cd portfolio.omarhosamcodes
```

2. Install dependencies:
```bash
bun install
```

3. Set up environment variables:
```bash
cp .env.example .env.local
# Edit .env.local with your database URL and other configurations
```

4. Set up the database:
```bash
bun run db:push
bun run db:seed
```

5. Start the development server:
```bash
bun run dev
```
